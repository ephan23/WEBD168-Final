The CSS3 rules and techniques I used in my project include:
- Integrated Fonts via Google Fonts
 (Local: Roboto, Web: Montserrat)
- Text shadowing for the title
- Multiple background images
- Linear Gradient
- Border/box effects
- RGBa and HSLa
- CSS Keyframe and @media

I'm satisfied with the outcome of my project because I was able to incorporate the CSS grid system to make the layout look smoother.
I was expecting to include more content but I wanted to keep it as simple as it is.

A challenge that I faced was with adding multiple backgrounds. For a while I was stuck because I couldn't figure out why one of the images wasn't showing on top of the main background image, but I soon found out that it wasn't prioritizing the image I wanted on top.
For design, the only challenge I had was thinking about where I should place each element on the page.

Overall, I am completely satisfied with the course because I was able to learn new concepts that I didn't know before, such as 3D transformations.
